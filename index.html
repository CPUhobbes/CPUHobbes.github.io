Raspi-Bot

App that enables users to remotely control a Raspberry Pi robot. Streams a live video feed so the user can move the robot beyond visual range. Users can add and remove multiple robots under their account.

The app is built using React and React-Router. MongoDB was used as the database.

The robot is built using Python, RPi.GPIO, pymongo, and pygame (for sound).

Movies for the Masses

Movies by the Masses is an web-based application that uses Twitter to find out how people are feeling about movies. The application uses movie, sentiment analysis, and Twitter API's to achieve this goal. Movies by the Masses works by having the user search for a movie title. After their input has been recorded, there is an AJAX call to the OMDB API (to get the movie information) and a call to the PHP script. The PHP script calls the Twitter API and returns an object of tweets based on the movie the user selected. The Sentiment140 API uses the these tweets (also called in the PHP script) and returns a value based on each tweet. After the sentiment analysis is completed, all the information is populated on the page.

React.js NY Times Article Search

Using React.js, a user can search the New York Times article database. The page will display articles that are searched as well as any articles saved to the database.

The app is broken into 3 children components inside the parent (main) component. The search component will update the results component after a query is made. If the user saves an article, the saved article component will update as well as store the article in the database.

Colors From Twitter

Post a color (only colors in the rainbow) to a Twitter account and the arduino will turn a LED that color. In addition the LCD attached to the arduino that will display the name of the color. Colors From Twitter @ Github

The project requires the use of an external webserver to host the .php files. The .php files that are used are the OAuth framework, a PHP Twitter authorization library (created by abraham@abrah.am, twitteroauth), and PHP/arduino interface file. The arduino uses 9 output pins (6 for the LCD and 3 for the LED) and uses the ethernet shield.
